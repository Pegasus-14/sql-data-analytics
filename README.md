# sql-data-analytics
Analyzes the data from the Data Warehouse. Specifically the data from the gold layer. For more information, please check the sql data warehousing project. The following scripts cover various analyses such as database exploration, measures and metrics, time-based trends, cumulative analytics, segmentation, and more. This repository contains SQL queries designed to help data analysts and BI professionals quickly explore, segment, and analyze data within a relational database. Each script focuses on a specific analytical theme and demonstrates best practices for SQL queries.
